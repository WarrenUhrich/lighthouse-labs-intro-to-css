IDs and Classes

What are IDs and Classes?

  • ID

    We can only use one ID per-element. Most-specific! Often used for one-time styles or JavaScript targeting. A particular ID can only be used once per-page! Symbol to use an ID in CSS: #

  • Class

    Can apply a class to many elements! Can apply multiple classes per-element. Can use a class as many times as you like per-page. More specific than an element, less specific than an ID. Symbol to use a class in CSS: .
